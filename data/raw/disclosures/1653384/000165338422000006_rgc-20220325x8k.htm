Item 5.05. Amendments to the Registrant’s Code of Ethics, or Waiver of a Provision of the Code of Ethics.

​

On March 22, 2022, Runway Growth Finance Corp.’s (the “Company”) Board of Directors approved an amended and restated Joint Code of Ethics (as amended, the “Code”). The Code was amended to include a section regarding the Company’s current policy on digital assets.
​

The foregoing description of the revisions reflected in the Code is qualified in its entirety by reference to the full text of the Code, a copy of which is filed as Exhibit 14.1 to this Current Report. A copy of the Code is also publicly available on the Company’s website at https://runwaygrowth.com.
